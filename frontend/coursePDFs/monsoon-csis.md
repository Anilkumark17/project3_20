## 01. Data Structures & Algorithms for Problem Solving

> 🔵 **CSIS Core**

| Field | Details |
|---|---|
| **Course Code** | `CS1.304` |
| **Credits** | 6 |
| **Instructor** | Kshitij Gajjar + Lini Thomas |
| **Department** | Computer Science and Engineering |
| **Semester** | Monsoon 2024 |

### 📋 Course Summary

A foundational PG course covering fundamental and advanced data structures—linked lists, trees, AVL trees, heaps, hash tables, tries—and core algorithms including recursion, searching, sorting, dynamic programming, and graph traversal, with emphasis on implementation and problem-solving.

### 🎯 Learning Outcomes

- Understand fundamental and advanced data structures including linked lists, BSTs, AVL trees, heaps, hash tables, tries, and suffix trees.
- Implement data structures and use them in abstract data type implementations.
- Devise solutions to programming challenges involving data structures and recursion.
- Understand algorithms including recursion, searching, hashing, dynamic programming, and graph traversal.
- Analyse algorithmic complexity and reason about program correctness using inductive proofs and invariants.
- Select appropriate data structures and algorithms for real-world problems and implement solutions.
